# yemreak's Homebrew Tap

Personal Homebrew formulas.

## Install

```
brew tap yemreak/tap
```

## Available

### openwith-cursor

Open code and text files in Cursor.

```
brew install openwith-cursor
```

Requires [Cursor](https://cursor.com).

See [release notes](https://github.com/yemreak/homebrew-tap/releases?q=openwith-cursor-).

---

**Note on source archives:** GitHub auto-attaches `Source code (zip)` and `Source code (tar.gz)` to every release. Those archives are the **tap-repo** content (this README plus formula definitions) — they are not the source of the installed tools. The installed binaries are closed-source.
